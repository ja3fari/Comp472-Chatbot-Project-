# AI Student Advising Chatbot

This project is a Python-based student advising chatbot. It loads a knowledge base from a CSV file and runs a conversation loop where users can type questions.

## Current Features

- Loads questions and answers from `knowledge_base.csv`
- Starts a chatbot conversation loop
- Allows the user to type questions
- Exits when the user types `quit`

## How to Run

```bash
python3 main.py
```
